## D) Laboratorioharjoitus
(terkarvinen.com, arvioitava laboratorioharjoitus)

Lista harjoituksessa tehdyistä asioista:

1. Linux-palvelimen asennus
- Debian/Ubuntu Linux asennettu virtualBox
- Hostname asetettu: p73

2. Käyttäjät
- juho (sudo käyttäjä)
- jaana (PHP kehityskäyttäjä)
- hakan (Python kehityskäyttäjä)

3. SSH etähallinta
- OpenSSH palvelin asennettu
- Etäkirjautuminen toimii
- Testattu: ssh juho@10.0.2.15

4. Tulimuuri
- UFW firewall käytössä
- HTTP ja SSH sallittu

5. WordPress palvelu
- Apache web server asennettu
- MariaDB tietokanta
- PHP asennettu
- WordPress asennettu
- VirtualHost konfiguroitu
- Sivusto: http://iot12.example.com
- DNS simuloitu /etc/hosts tiedostolla

6. PHP kehitysympäristö
- Apache userdir käytössä
- Käyttäjälle jaana oma kehityssivu
- Osoite: http://localhost/~jaana
- PHP sivu toimii

7. Python kehitysympäristö
- Käyttäjälle hakan testiohjelma
- Ohjelma: hello.py
- Testattu: python3 hello.py

8. Oma komento
- whenwhere skripti tehty
- Tulostaa IP-osoitteen ja kellonajan
- Komento toimii kaikille käyttäjille

9. Järjestelmän testaus
- curl testit web-sivuille
- SSH testattu
- Python ohjelma testattu
- whenwhere komento testattu

10. Dokumentointi
- lab.txt raportti
- ruutukaappaus testituloksista
- järjestelmän tiedot pakattu lab.tar.gz
