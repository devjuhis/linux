# Raportti Linuxin virtuaalikoneen asentamisesta

1. latasin oracle virtual boxin ja iso kuvan
2. kun aloin luomaan uutta virtuaalikonetta ja lisäsin iso kuva näytti 32 bittistä versiota vaikka pitäisi olla 64.
katsoin chatgpt:stä mikä mättää ja se ohjasi ohjauspaneeliin ottamaan windowsin hyper-v, virtual machine platform, Windows hypervisor platform, Windows sandboxin pois päältä.
3. sanoo että this kernel requiresan x86-64cpu, but only detected an i686cpu. Unable to boot - please ude a kernel appropriate for your cpu
4. Muistin eheys pois päältä (chatgpt)
5. uudelleen käynnistymisen jälkeen virtuaalikone alkoi toimimaan
6. asensin debianin (Tero Karvinen)
7. Tein käyttäjän ja kirjauduin (Tero karvinen)
8. ajoin terminaalikomennot, joissa päivitin kaikki ohjelmat (Tero karvinen)
